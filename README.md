# Rubrica Punto 1 - Gestión de Estado

Este proyecto contiene información sobre la gestión de estado en React.js.

- **Carpeta `Ejemplos`**: Contiene ejemplos prácticos en formato `.jsx` sobre el uso de `useState`, Redux y Zustand.
- **Archivo `PREGUNTAS.txt`**: Contiene las respuestas detalladas a las preguntas teóricas sobre State, manejadores de estado, Redux y Zustand.
